Tentang Aplikasi

Sistem Prediksi Distribusi Pekerjaan Menggunakan Metode Monte Carlo

Deskripsi Proyek

Aplikasi ini dikembangkan untuk memprediksi distribusi jenis pekerjaan di Kota Bandung menggunakan metode Monte Carlo. Metode ini menggunakan simulasi acak untuk menghasilkan prediksi statistik dengan mempertimbangkan ketidakpastian data historis.

Fitur Utama:
  • Prediksi Monte Carlo: Menggunakan simulasi acak untuk prediksi distribusi pekerjaan
  • Confidence Interval: Menampilkan rentang kepercayaan 95% untuk setiap prediksi
  • Visualisasi Interaktif: Chart dan dashboard real-time untuk analisis data
  • Export Data: Hasil prediksi dapat diexport ke format Excel untuk analisis lebih lanjut
  • Dashboard Analytics: Monitoring dan analisis data historis dengan berbagai chart

Metodologi

Metode Monte Carlo

Metode Monte Carlo adalah teknik komputasi yang menggunakan pengambilan sampel acak berulang untuk mendapatkan hasil numerik. Dalam aplikasi ini, metode ini diterapkan untuk:

1
Analisis Data Historis

Menghitung rata-rata dan standar deviasi pertumbuhan

2
Simulasi Acak

Generate ribuan skenario pertumbuhan menggunakan distribusi normal

3
Prediksi Statistik

Hitung nilai rata-rata dan interval kepercayaan dari semua simulasi

4
Visualisasi Hasil

Tampilkan prediksi dengan confidence interval dalam chart interaktif
